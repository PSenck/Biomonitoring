class Experiment.Experiment(path, exp_nr, meta_path='metadata.xlsx', types=['off', 'on', 'CO2'], prefix=['offline/offline_', 'online/online_', 'CO2/CO2_'], suffix=['.csv', '.CSV', '.dat'], index_ts=[0, 0, 0], read_csv_settings=[{'sep': ';', 'encoding': 'unicode_escape', 'header': 0, 'usecols': None}, {'sep': ';', 'encoding': 'unicode_escape', 'decimal': ',', 'skiprows': [1, 2], 'skipfooter': 1, 'usecols': None, 'engine': 'python'}, {'sep': ';', 'encoding': 'unicode_escape', 'header': 0, 'skiprows': [0], 'usecols': [0, 2, 4], 'names': ['ts', 'CO2', 'p']}], to_datetime_settings=[{'format': '%d.%m.%Y %H:%M', 'exact': False, 'errors': 'coerce'}, {'format': '%d.%m.%Y  %H:%M:%S', 'exact': False, 'errors': 'coerce'}, {'format': '%d.%m.%Y %H:%M:%S', 'exact': False, 'errors': 'coerce'}])

Bases: object

A class for creating an Experiment object with preprocessed datasets containing offline(HPLC, bio dry mass), online (MFCS), CO2(Bluesense) data. The class assumes that there is a common path where the whole data is present within.

Parameters

  • path (str) – Path of overarching directory, the metadata.xlsx file and the directories containing the data (e.g. online, offline and CO2) must be within this directory.

  • exp_nr (int) – Number of Experiment in the filename (e.g. offline_3). Number in filename and in metadata must match and should correspond to the same experiment.

  • meta_path (str) – metadata file name within path.

  • types (list with strings) – List with measurement data types.

  • prefix (list with strings) – List with file prefix names within path, must match with types.

  • suffix (list with strings) – List with data endings (data types), must match with types

  • index_ts (list with int's) – List with position of the index column of the raw data.

  • read_csv_settings (list with dicts) – List with pd.read_csv settings to read the correspondig raw data to pd.DataFrame.

  • to_datetime_settings – List with settings to convert Timestamp column in raw data to pd.Timestamp

Returns

Experiment object contaning the measurement data.
